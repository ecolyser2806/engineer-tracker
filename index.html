⚙ Setup Google Sheets
One-time configuration — takes about 5 minutes

Connect to Google Sheets

Follow these steps once. After setup, all data saves automatically to your Google Sheet and can be viewed or edited by anyone you share it with — from any device.

1
Open Google Sheets and create a new blank spreadsheet. Name it Ecolyser Incentive Tracker.
2
In that sheet, click Extensions → Apps Script. A new tab opens with a code editor.
3
Delete everything in the editor. Open the google_apps_script.js file you downloaded and paste its entire contents. Click Save (Ctrl+S). Name the project EcoTrack.
4
Click Deploy → New Deployment. Set Type = Web App. Set Execute as = Me. Set Who has access = Anyone. Click Deploy. Google will ask you to authorise — click Allow.
5
Copy the Web App URL that appears (starts with https://script.google.com/macros/s/...). Paste it below and click Connect.
📝 After Connecting — How to Edit Data
• Open your Google Sheet anytime from any device or browser
• You will see 3 tabs: Engineers, Monthly Records, and Daily Logs
• Click any cell and edit directly — changes are saved instantly in Google Sheets
• Click 🔄 Refresh in this tracker to reload the latest data from the Sheet
• Share the Google Sheet with your team using the Share button in Google Sheets
• To fix an incentive amount — edit the value fields in Monthly Records tab; the tracker will recalculate on next load
